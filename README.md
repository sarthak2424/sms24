An sms tool for Andriod, Linux and Windows

## Install & Run (Linux & Andriod)
```
git clone https://github.com/MetaChar/Tortuga
cd Tortuga
pip install smtplib
python2 sms.py
```

## Install & Run (Windows)
```
git clone https://github.com/MetaChar/Tortuga
cd Tortuga
pip install smtplib
pip install colorama
python2 sms_windows.py
```
